"""
Overrides the call operator to implement the Singleton design pattern.

This method intercepts the class instantiation. It checks if an instance of the
class already exists in the `_instances` dictionary. If not, it creates a new
instance by calling the superclass's `__call__` method and caches it. Subsequent
calls return the previously created instance.

Args:
    *args: Variable length argument list passed to the class constructor.
    **kwargs: Arbitrary keyword arguments passed to the class constructor.

Returns:
    The single, persistent instance of the class.
"""
